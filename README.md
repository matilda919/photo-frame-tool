# 照片加边框工具

一个简单易用的在线照片处理工具，为照片添加白色边框和水印。

## 功能特性

- 📷 添加白色边框
- 📐 自动调整为9:16竖版画幅
- 📱 照片自动居中显示
- 🏷️ 自动读取拍摄设备信息
- ✍️ 自定义拍摄者名称
- 💾 一键下载处理后的照片

## 使用方法

1. 点击上传区域选择照片
2. 输入拍摄者名称
3. 点击"生成照片"按钮
4. 预览效果并下载

## 支持设备

- iPhone/iPad (Safari浏览器)
- Android手机 (Chrome浏览器)
- 桌面浏览器

## 技术栈

- HTML5
- CSS3
- JavaScript (Canvas API)
- EXIF数据读取
